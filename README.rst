============================
Question2Answer Accept Rate
============================

-----------
Description
-----------
This is a plugin for **Question2Answer** that adds accept rate to question user meta.

--------
Features
--------
- optional shading or custom css for text
- tooltip explanation on hover over
- text customizable using language files

------------
Installation
------------
#. Install Question2Answer_
#. Get the source code for this plugin from github_, either using git_, or downloading directly:

   - To download using git, install git and then type 
     ``git clone git://github.com/NoahY/q2a-accept-rate.git accept-rate``
     at the command prompt (on Linux, Windows is a bit different)
   - To download directly, go to the `project page`_ and click **Download**

#. navigate to your site, go to **Admin -> Plugins** on your q2a install and select the '**Show accept rate below user meta**' option, then '**Save**'.
